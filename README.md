 Sistema Embebido con ESP32, Pantalla Táctil y Sensor DHT11

 ¿Qué es esto?

Este proyecto es un **sistema interactivo** desarrollado para un microcontrolador ESP32. Incluye una interfaz gráfica con pantalla táctil ILI9341, un sensor de temperatura y humedad DHT11, un lector de tarjetas SD y un juego tipo Snake integrado.

 Objetivo del proyecto

Demostrar el funcionamiento de un sistema embebido que integra múltiples periféricos mediante una interfaz amigable para el usuario, con navegación táctil y retroalimentación visual y sonora.
